# 🧾 POS Billing & GST System (C++)

## 📌 Project Overview
The **Point of Sale (POS) Billing & GST System** is a console-based C++ application that simulates a retail billing system. It allows users to select products, calculate GST automatically, generate invoices, and display a daily sales summary.

This project demonstrates how programming can be used to automate retail billing operations while ensuring accurate tax calculations.

---

## 🧩 Problem Statement
Retail stores require efficient billing systems to calculate product prices, apply taxes, and generate invoices quickly. Manual billing can lead to calculation errors and inefficiencies.

This project develops a **POS Billing System** that automates the billing process and calculates **GST (Goods and Services Tax)** accurately.

---

## 🔄 System Workflow
1. The system stores a list of products with price and GST rate.
2. The user enters the **product ID** to select a product.
3. The user enters the **quantity** of the product.
4. The system calculates:
   - Product subtotal
   - GST amount
   - Final price
5. After all products are entered, the system generates a **final invoice**.
6. The system displays the **total sales and GST collected**.
